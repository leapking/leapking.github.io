检查蓝牙协议版本

BlueZ是Linux官方的蓝牙协议栈。我们可以通过BlueZ提供的接口，进行丰富的蓝牙操作。查看Raspbian中BlueZ版本。

低版本的BlueZ对低功耗蓝牙（早期蓝牙耗电较高，诺基亚改良了低功耗的蓝牙协议）的支持有限。如果你的使用版本低于5.43，先升级BlueZ。

操控蓝牙设备

在Raspbian中，基本的蓝牙操作可以通过bluez提供的bluetoothctl命令进行。命令运行后将开启一个新的命令行环境。

FAQ：
[bluetooth]# scan on
Failed to start discovery: org.bluez.Error.NotReady

方法一：

看蓝牙状态中有rfkill报错。RFKill 是Linux内核中的一个子系统，它可提供一个接口，在此接口中可查询、激活并取消激活计算机系统中的无线电传输。
当取消激活传输时，可使其处于可被软件重新激活的状态（ 软锁定 ）或软件无法重新激活的位置（ 硬锁定 ）。

使用命令 rfkill list 获得设备列表，每个都包含与之关联的索引号 ，从 0 开始。
