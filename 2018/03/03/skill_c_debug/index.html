C语言的一些调试技巧

编译

  1. gdb禁用优化，在编译是加入 ‘-O0’ 选项
  2. gdb查看宏定义，在编译是加入 ‘-g3’ 选项
  3. gdb调试时局部变量不可查看时，在编译是加入 ‘-gstabs+’选项
    比如:
    g++ -g -Wall -O0 -g3 -gstabs+ -o main.o main.cpp
    或：
    ./configure … CFLAGS=’-O0 -g3 -gstabs+’
