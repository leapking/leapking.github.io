一些重要的系统表

系统表 功能
pg_authid 存储用户信息
pg_tablespace 存储表空间
pg_namespace 存储schema
pg_database 存储数据库信息
pg_class 存储Relation对象的元信息
pg_index 存储索引的元信息
pg_attribute 存储列类型的元信息
pg_constraint 存储约束的元信息

如何查看PostgreSQL正在执行的SQL

procpid：进程id
start：进程开始时间
lap：经过时间
current_query：执行中的sql
怎样停止正在执行的sql
SELECT pg_cancel_backend(进程id);
或者用系统函数
kill -9 进程id;
