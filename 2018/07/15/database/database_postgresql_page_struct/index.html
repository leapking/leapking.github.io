数据页面分析

页面头总共20个字节，之后是元组的槽位，之后是空闲空间，之后是元组值，之后是special空间（只有index页面才有）

(lower - 20 ) / sizeof(item) 就是该页面的元组个数了

插入元组时先在最后一个item后添加一个新的item，lower向上增长sizeof(item)

元组值是从页面尾部倒着占用空间的，最后一个元组的开始位置也是upper指向的位置
